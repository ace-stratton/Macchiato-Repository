==========================================================================================
    Quick-start guide
==========================================================================================
1. Unzip macchiato in a folder in your project;
2. Edit the files in the unzipped models folder;
    --> FIDL interfaces are described under models\org\endurosat\interfaces;
    --> Deployment information such as whether you want a client or server generation is specified in models\org\endurosat\deploy\deploy_all_default.fdepl;
3. Run start_gen.bat;
4. Find your generated files in the newly created src-gen-c, src-gen-py and src-gen-js folders;

==========================================================================================
    I have a question?
==========================================================================================

It is html based documentation. You can open it by executing 'open_docs.bat'
